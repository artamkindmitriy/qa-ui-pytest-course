# UI Automation & PyTest Sandbox Project 🧪

Репозиторий содержит практические задачи и тесты, написанные в процессе изучения основ автоматизации тестирования UI и базовых возможностей PyTest на Python.

## 🛠️ Стек технологий:
* **Язык программирования:** Python
* **Тестовый фреймворк:** PyTest
* **Автоматизация UI:** Selenium WebDriver (с использованием WebDriver Manager и ActionChains)

---

## 📁 Содержание проекта:

### 1. Основы PyTest (`tests/pytest_basics/`)
Практика использования ключевых возможностей фреймворка PyTest:
* **Паттерн AAA (Arrange, Act, Assert):** Написание понятных и структурированных тестов.
* **Фикстуры (Fixtures):** Создание, переиспользование и очистка тестового окружения (включая работу с `yield` и удаление временных файлов).
* **Параметризация:** Оптимизация тестов для проверки различных наборов данных (включая проверку граничных значений и валидаторы).
* **Группировка:** Организация тестов в тестовые классы (`TestClass`).

### 2. Тестирование UI с Selenium (`tests/selenium_ui/`)
Автоматизация взаимодействия с веб-интерфейсами на тренировочных площадках (`saucedemo.com`, `the-internet.herokuapp.com`):
* Навигация по страницам и проверка заголовков сайтов (`driver.back`, `driver.forward`, `driver.title`).
* Поиск элементов по `id`, `name`, `class name` и `xpath`.
* Работа с веб-формами: ввод текста, очистка полей, чтение атрибутов (`get_attribute`).
* Взаимодействие с элементами интерфейса: клики, чекбоксы (`is_selected`).
* Эмуляция действий мыши через `ActionChains` (наведение курсора).
